## Dependencies

- [x] [Publish](https://github.com/johnsundell/publish) - A static site generator for Swift developers

## Build locally

```sh
git clone --recursive git@github.com:tapsnapapp/tapsnap.site.git
cd tapsnap.site/
open Package.swift
```
